Help

Load Custom ROM

  • 1: Get dropbox from the AppStore Link
  • 2: Get a unzipping tool from the AppStore and unzip the zip file Link
  • 3: Download a rom file Link
  • 4: Unzip it
  • 5: Click on the .gba file and open it in Dropbox
  • 6: Go to iGBA and click on the Add Rom button and choose Dropbox
  • 7: Select the .gba file that you have on your Dropbox
